  # Install
  1. Clone project
   
    git clone https://github.com/omidKinfar/my-react-todo-app.git
    
  2. Install dependencies

    npm install
        
  3. Run project

    npm start
    
  4. Run json server

    npx json-server -w db.json -p=3050
